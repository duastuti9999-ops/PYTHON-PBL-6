🌊 Urban Water Demand Prediction
📌 Project Overview

This project predicts urban household water demand using Machine Learning techniques.
The system analyzes household and environmental factors to estimate water consumption patterns.

The project helps in:

Efficient water resource management
Predicting future water usage
Identifying high-consumption households
Supporting smart city water planning
📂 Dataset Features

The dataset contains the following columns:

household_size – Number of people in the house
seasonal_index – Seasonal variation factor
garden_area – Garden size
number_of_bathrooms – Bathrooms count
children_count – Number of children
elderly_count – Number of elderly members
appliance_count – Water-using appliances count
temperature – Environmental temperature

Target Variable:

water_demand

🤖 Machine Learning Models Used
1. Linear Regression

Predicts continuous water demand values.

2. Decision Tree

Predicts demand using decision-based splitting.

3. K-Means Clustering

Groups households based on water usage behavior.

The project uses:
